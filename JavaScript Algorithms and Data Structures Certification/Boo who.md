**Boo who**

**CHALLENEGE**

Check if a value is classified as a boolean primitive. Return true or false.

Boolean primitives are true and false.

```
function booWho(bool) {
  return typeof bool === "boolean";
}

booWho(null);

```

**SOLUTION**

```
function booWho(bool) {
  return typeof bool === "boolean";
}

booWho(null);
```
